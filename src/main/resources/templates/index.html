WorkShop DevOps

Site exemplo de uma pipeline devops

Atenção! Siga as instruções abaixo para realizar o deploy.

Instruções

  • Fazer um fork do repositorio https://github.com/
  • Pode utilizar uma ide para realizar as alterações
  • Realizar a integração com o Travis-ci
  • Realizar a integração de deploy do Heroku
  • Implementar o teste
  • Implementar os procedimentos para passar nos testes

Integração Contínua CI

  • Todos os testes são realizados todas as vezes antes de ser colocado em produção
  • Caso algum teste falhe, o sistema em produção não é alterado
    • Teste de Unidade
    • Teste funcional
    • Teste de Integração
    • Teste de Segurança

Entrega Contínua

  • O produto está sempre sendo entregue
  • Maior "agilidade" no processo de Verificação & Validação
